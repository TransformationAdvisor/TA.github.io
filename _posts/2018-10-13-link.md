If for example the profile name is AppSrv010 and it is under /root/was_profile
Create a symbolic link to the profile as follows :
{% highlight ruby %}
ln -s /root/was_profile /opt/IBM/WebSphere/AppServer/profiles/AppSrv010
{% endhighlight %}
